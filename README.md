# INT MAIN VIBE

**AI & Tech Brief — @wheresmaicol**

A brutalist parallax online magazine covering the AI & Tech space. Five editorial features tracking the biggest stories of 2026:

1. **The Model Race** — New model releases & capability jumps
2. **The Disappearing First Job** — AI & jobs / the economy
3. **The $700B Bet** — AI infrastructure & capital
4. **Who's In Charge?** — AI ethics & safety concerns
5. **Can You Trust What You See?** — Deepfakes & synthetic media

## Stack
- Static HTML / CSS / Vanilla JS
- Hosted on GitHub Pages
- No frameworks, no build tools
- Fonts: Archivo (display) + Space Mono (mono)

## Live Site
Once GitHub Pages is enabled → `https://lmminier.github.io/INT-MAIN-VIBE/`

## Enable GitHub Pages
1. Go to **Settings → Pages**
2. Set Source to **GitHub Actions**
3. The workflow at `.github/workflows/pages.yml` auto-deploys on every push to `main`.
