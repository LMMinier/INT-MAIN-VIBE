# INT MAIN VIBE

**AI & Tech Brief — @wheresmaicol**

A brutalist parallax online magazine covering the AI & Tech space. Seven editorial features tracking the biggest stories of 2026:

1. **Fable 5 & Mythos** — The model race / lead story
2. **$650 Billion and Counting** — AI infrastructure & capital
3. **The Disappearing First Job** — AI & jobs / the economy
4. **Who's In Charge?** — AI ethics & governance
5. **Can You Trust What You See?** — Deepfakes & synthetic media
6. **72 Hours, Then Gone** — The Fable 5 suspension (breaking)
7. **Not Just AI** — The whole stack: chips, cybersecurity, space, digital money

Every data panel is a **native HTML/CSS/SVG infographic** (bar charts, KPI callouts,
donut charts) — no stock images, no flat JPEGs. They're crisp at any resolution,
accessible (each carries an `aria-label`), and load instantly.

## Stack
- Static HTML / CSS / Vanilla JS
- Hosted on GitHub Pages
- No frameworks, no build tools
- Fonts: Archivo (display) + Space Mono (mono)

## Live Site
Once GitHub Pages is enabled → `https://lmminier.github.io/INT-MAIN-VIBE/`

## Enable GitHub Pages
1. Go to **Settings → Pages**
2. Set Source to **GitHub Actions**
3. The workflow at `.github/workflows/pages.yml` auto-deploys on every push to `main`.
